# 在线考试系统测试脚本使用说明

## 概述

本项目提供了两个测试脚本，用于验证在线考试系统的功能是否健全：

1. **test_system.py** - 完整功能测试脚本
2. **test_quick.py** - 快速测试脚本

## 前置条件

### 1. 启动 Django 后端服务器

```bash
cd exam_system
python manage.py runserver 8010
```

确保服务器在 `http://localhost:8010` 上正常运行。

### 2. 安装依赖

测试脚本需要 `requests` 库，如果未安装请运行：

```bash
pip install requests
```

## 常见问题

### 1. 无法连接到服务器

**错误信息**：
```
✗ 无法连接到服务器 http://localhost:8010/api
```

**解决方案**：
- 确保 Django 服务器正在运行
- 检查端口 8010 是否被占用
- 确认服务器地址配置正确（修改脚本中的 `BASE_URL`）

### 2. 用户已存在

**错误信息**：
```
✗ 注册用户: test_admin (admin)
  错误: 用户名已存在
```

**解决方案**：
- 这是正常现象，测试脚本会继续使用已存在的用户
- 如需重新测试，可以先删除数据库中的测试用户

### 3. 权限错误

**错误信息**：
```
✗ 创建试卷 (teacher)
  错误: 无权限访问
```

**解决方案**：
- 检查用户角色是否正确
- 确认后端权限验证逻辑

### 4. 数据库错误

**错误信息**：
```
✗ 添加题目 (teacher)
  错误: 数据库操作失败
```

**解决方案**：
- 检查数据库连接是否正常
- 确认数据库表结构是否正确
- 查看后端日志获取详细错误信息

## 自定义测试

### 修改服务器地址

如果服务器不在 `localhost:8010`，修改脚本中的 `BASE_URL`：

```python
BASE_URL = "http://your-server:port/api"
```

### 修改测试超时时间

如果服务器响应较慢，修改 `TEST_TIMEOUT`：

```python
TEST_TIMEOUT = 60  # 增加到 60 秒
```

### 添加自定义测试

参考现有的测试方法，添加新的测试：

```python
def test_custom_feature(self, role: str = "admin") -> bool:
    """测试自定义功能"""
    test_name = f"测试自定义功能 ({role})"
    try:
        response = self.session.post(
            f"{self.base_url}/your-endpoint/",
            headers=self.get_headers(role),
            json={"param": "value"}
        )
        return self.assert_response(response, 200, test_name)
    except Exception as e:
        self.result.add_fail(test_name, str(e))
        self.print_test(test_name, "FAIL", str(e))
        return False
```

## 持续集成建议

可以将测试脚本集成到 CI/CD 流程中：

```yaml
# .github/workflows/test.yml 示例
name: Run Tests

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: Set up Python
        uses: actions/setup-python@v2
        with:
          python-version: '3.10'
      - name: Install dependencies
        run: |
          pip install requests
      - name: Start Django server
        run: |
          cd exam_system
          python manage.py runserver 8010 &
      - name: Run tests
        run: |
          python test_quick.py
```
